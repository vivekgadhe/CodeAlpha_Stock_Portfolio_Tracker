 📈 CodeAlpha_Stock_Portfolio_Tracker

A simple Python project to calculate the **total investment value** of a stock portfolio using hardcoded stock prices.  
This project demonstrates **dictionaries, input/output, arithmetic operations, and file handling** with results saved in TXT/CSV formats.

---

🚀 Features
- User inputs stock symbols and quantities.
- Predefined dictionary of stock prices (`prices.py`).
- Calculates total investment value using modular functions (`portfolio.py`).
- Optionally saves results in `.txt` or `.csv` for structured output.

---

📂 Project Structure
```
CodeAlpha_Stock_Portfolio_Tracker/
│
├── main.py              # Entry point of the program
├── portfolio.py         # Function to calculate total investment
├── prices.py            # Hardcoded dictionary of stock prices
```
