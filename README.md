# 📈 Stock Portfolio Tracker

A simple Python program to calculate the **total investment value** of a stock portfolio.

## 🚀 Features
- User inputs stock symbols and quantities.
- Hardcoded dictionary of stock prices (`prices.py`).
- Calculates total investment using basic arithmetic.
- Optionally saves results to a `.txt` file.

## 🛠️ Tech Concepts
- **Dictionary** for stock prices  
- **Input/Output** for user interaction  
- **Arithmetic** for investment calculation  
- **File Handling** for saving results  

## ▶️ How to Run
```bash
python main.py
